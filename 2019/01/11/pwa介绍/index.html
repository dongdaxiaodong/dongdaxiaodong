pwa介绍

作为一个前端开发者，我本身对Android和ios开发并不熟悉，这也就意味着，我没法做出广泛意义上的’app’, web app的使用体验到如今9012年也没办法追上native app.但，最近 一项新的技术 progressive web app 进入了我的世界，于是，作为一个前端开发者的我也可以开发出媲美原生app的产品了。

渐进式网页应用(pwa)的核心为三项技术.1 manifest 2. service worker 3.消息推送

网页应用和原生应用的一个很大的区别在于，手机端 web app的地址栏，导航栏导致在外观上和native app有很大差别，manifest实际上是一个json文件，里面包含这个app用于安装的各项信息，正是有了它的存在，我们的pwa可以安装在手机桌面，而且还有它相应的图标。

网页应用和原生应用还有一个大的区别是，网页应用不能离线使用，在网络不好或者断网的情况下我们的应用就相当于摆设，而service worker这一缓存技术解决了这一难题，使用Cache用于缓存app需要的各项数据，且这些数据是什么，怎么用完全由开发者决定!

最后一点，pwa还实现了消息推送功能，这也意味着 我们的app也能够提高用户粘性!

我自己写了一个pwa demo 项目地址 可以作为参考，应用地址这是一个todo应用，用户的数据使用localstorage存储在本地，方便好用，记得添加到桌面哦

作为一个开发者，其实微信小程序也为我们开发应用提供了一个渠道，但我并不为其所动。微信小程序始终是一个封闭的环境，web的本质是开发，共联的，所以 pwa才是未来!!!
